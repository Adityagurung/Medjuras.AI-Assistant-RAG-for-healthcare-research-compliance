# Grafana (monitoring only)

This folder contains **Grafana provisioning and dashboards** for the MedJuras stack.

- `provisioning/` — datasources and dashboard provider config (mounted by `docker-compose.yaml`)
- `dashboards/` — `medjurisrag.json` dashboard definition
- `assets/` — architecture diagram (optional reference)

Application code lives at the repo root: `app/`, `config/`, `notebooks/`, `scripts/`.
